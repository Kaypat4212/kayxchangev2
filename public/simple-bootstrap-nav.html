📱 Mobile Test Instructions
  1. Resize your browser window to mobile size (less than 992px)
  2. The navbar will automatically collapse to show the hamburger menu
  3. Click the hamburger menu (three lines) to expand the navigation
  4. Click on any navigation link or outside the menu to close it

This uses pure Bootstrap 5 functionality - no custom JavaScript required!
